What we're looking for with this challenge

The most important thing for becoming a new member on this team isn't prior ability, you don't have to have to be a coding wizard! You just have to be willing to pick up the skills necessary when presented with a problem.

The challenges have been created accordingly. With enough perserverance, you should be able to complete them with only basic programming experience. The most complex that the math gets should be high-school level trigonometry.



Please complete this challenge before:

September 30 2020

The challenge is contained in a github repo here. To get started, just clone the repo and read the README.

(Note that this is a new software challenge for this term. If you've already completeted the old software challenge, we will accept that as well!)
